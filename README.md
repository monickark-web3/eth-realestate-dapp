# eth-realestate-dapp

**Real Estate Decentralized Application** built on Ethereum. Buy, sell, and list properties on the blockchain with full transparency and no intermediaries.

---

## ✨ Features

- List properties for sale
- Buy properties using cryptocurrency
- View all listed & owned properties
- Secure ownership transfer via smart contracts
- Wallet connection (MetaMask)
- Responsive and modern UI

---

## 🛠 Tech Stack

- **Blockchain**: Solidity, Hardhat, Thirdweb
- **Frontend**: Next.js, React, JavaScript
- **Styling**: Material-UI
- **Web3**: Ethers.js

---

## 📁 Project Structure

```
eth-realestate-dapp/
├── contracts/              # Smart Contracts (Hardhat)
├── client/                 # Next.js Frontend
├── thirdweb-contracts/     # Thirdweb contracts
└── README.md
```

---

## 🚀 Quick Start

### 1. Clone the repo
```bash
git clone https://github.com/monickark-web3/eth-realestate-dapp.git
cd eth-realestate-dapp
```

### 2. Install Smart Contracts
```bash
cd contracts
npm install
npx hardhat compile
```

### 3. Run Frontend
```bash
cd client
npm install
npm run dev
```

Open http://localhost:3000
